Choose your database

The platform supports three databases. You can connect more later — but most teams start with one.

PostgreSQL Recommended
Open source · row-level security · realtime · vector search · 95% feature parity with Supabase
Use for: Most apps
Strengths: Versatile, ACID, JSON
Self-host: Yes (included)
Microsoft SQL Server
Enterprise standard · Entra ID native · existing licensing leverage
Use for: Enterprise migrations
Strengths: AD/Entra, T-SQL
Self-host: Yes
MongoDB
Document-first · flexible schemas · change streams · for content-heavy apps
Use for: Document data
Strengths: Schema flex
Self-host: Yes
Need more than one? You can add additional databases later from Settings → Databases. The platform's hexagonal architecture lets you mix all three in the same project (different services, different stores).
